android studio中找不到“sync the project with gradle files”按钮时的操作

android studio中找不到“sync the project with gradle files”按钮时的操作

  1. 打开Gradle工具窗口 View -> Tool Windows -> Gradle
  2. 使用开始宏录制 Edit -> Macros -> Start Recording Macro
  3. 在Gradle工具窗口中按下Refresh all Gradle projects按钮
  4. 用宏停止宏录制 Edit -> Macros -> Stop Recording Macro
  5. 用名字保存宏 Gradle sync
  6. 在IntelliJ IDEA工具栏上用鼠标右键按下并选择 Customize Menus and Toolbars...
  7. 在Main menu -> Edit -> Macros -> Macro Actions找到您的Gradle sync宏，选择它，然后按Edit Action Icon...
  8. 下载此图标同步项目与Gradle文件图标（从文档http://developer.android.com/google/play-services/setup.html），并在IDEA对话框窗口中选择此图标
  9. 现在在组中Menus and Toolbars选择最后一项Main Toolbar，按Add After...并选择All actions -> Macros -> Gradle sync
  10. 现在你有Sync Project with Gradle Files按钮！
